Azure Entra ID / Microsoft 365 Help Desk Simulation Lab
Overview

This project simulates a real-world IT Help Desk environment using Microsoft Entra ID and Microsoft 365 Admin Center. The purpose of this lab is to demonstrate practical skills in user account administration, troubleshooting, and incident resolution within an enterprise identity management system.

The lab focuses on common IT support scenarios such as password resets, access issues, licensing problems, and account status verification.

Objectives

The main objectives of this project are:

Simulate real IT Help Desk ticketing scenarios
Perform user account management in Microsoft Entra ID
Troubleshoot Microsoft 365 access and licensing issues
Investigate and resolve user authentication problems
Document technical incidents in a structured and professional manner
Demonstrate IT support workflow and problem-solving skills
Tools and Technologies Used
Microsoft Entra ID (Azure Active Directory)
Microsoft 365 Admin Center
Microsoft 365 Developer Tenant
Windows Notepad (for ticket documentation)
GitHub (for project documentation and version control)


Help Desk Tickets
Ticket 001 – Password Reset

Issue:
User was unable to access their account due to a forgotten password.

Resolution:
Password was reset using Microsoft Entra ID, and access was restored successfully.

Ticket 002 – Security Group Access Issue

Issue:
User reported missing access to required resources.

Resolution:
User was added to the appropriate security group, restoring access to the required systems.

Ticket 003 – Microsoft 365 Licensing Issue

Issue:
User was unable to access Microsoft Outlook and Microsoft Teams.

Investigation:
Reviewed assigned licenses within Microsoft 365 Admin Center.

Findings:
No available licenses were present in the tenant.

Resolution:
Documented licensing limitation and confirmed requirement for license allocation within the environment.
