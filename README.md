🚀 Azure Entra ID / Microsoft 365 Help Desk Lab
📌 Overview

This project simulates a real-world IT Help Desk environment using Microsoft Entra ID (Azure Active Directory) and Microsoft 365 Admin Center.

The goal of this lab was to demonstrate practical skills in:

User account management
Password reset and access troubleshooting
Security group and permission management
Microsoft 365 licensing investigation
Account status verification and support documentation
IT ticket handling and resolution workflows
🧰 Tools & Technologies Used
Microsoft Entra ID (Azure Active Directory)
Microsoft 365 Admin Center
Windows Notepad (ticket documentation)
GitHub (project documentation)
🎯 Project Objectives
Simulate real IT support ticket scenarios
Practice user administration in Entra ID
Troubleshoot common Microsoft 365 access issues
Document incidents professionally like a Help Desk Technician
Build a portfolio-ready IT support project
